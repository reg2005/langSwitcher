Built for productivity

Everything you need to stop wasting time retyping text in the wrong layout.

⚡

Instant Conversion

Press a hotkey and text is converted in milliseconds through clipboard-based replacement. Works in any app.

🧠

Smart Conversion

No need to select text. Greedy Line mode auto-detects where wrong layout begins and converts only what's needed.

⌨

Double Shift Hotkey

Default hotkey: press Shift twice quickly. Or record any custom shortcut in settings.

📊

Conversion Log

Every conversion is logged to a local SQLite database. Rate as correct/incorrect. Export as JSON for ML training.

🌍

5 Layouts

English, Russian, German, French, Spanish. Auto-detected from system keyboard settings.

🔒

Privacy First

No data leaves your Mac. No analytics, no telemetry, no network access. Fully offline. Open source under MIT.

📌

Punctuation Preserved

Symbols like ?, !, / stay unchanged during conversion. Only layout-specific characters are transformed.

🛠️

Zero Dependencies

Pure Swift, no external libraries, no dictionaries. Purely algorithmic detection based on physical key positions.

✅

93 Unit Tests

Comprehensive test suite covering layout mapping, gibberish detection, greedy boundary detection, and conversion pipeline.

Smart Conversion

No more manually selecting text. LangSwitcher detects what needs converting automatically.

ModeBehaviorExample
Greedy Line (default)Selects to line start, scans right-to-left to find where wrong layout begins, converts only the wrong portionghbdtn rfr ltkf → привет как дела
Last WordAuto-selects only the last typed word, converts if it looks like wrong layout...lheu → ...друг
DisabledOnly works with explicit manual text selectionSelect text → ⇧⇧ → converted
