Откройте настройки, нажав на иконку LangSwitcher в строке меню → **Настройки**.

## Вкладка «Основные»

- **Режим умного преобразования** — выберите между «Жадная строка», «Последнее слово» или «Отключено». Подробнее см. [Умное преобразование](/ru/getting-started/smart-conversion).
- **Запуск при входе** — автоматически запускать LangSwitcher при входе в систему.
- **Показывать уведомления** — показывать краткое уведомление после каждого преобразования.
- **Язык** — переключение интерфейса между английским и русским.

![Вкладка «Основные»](/screenshots/general.png)

## Вкладка «Горячая клавиша»

- **Двойной Shift** — использовать стандартную горячую клавишу.
- **Пользовательская** — нажмите «Записать» для настройки любой комбинации.

Подробнее см. [Настройка горячей клавиши](/ru/getting-started/hotkey).

## Вкладка «Раскладки»

Показывает, какие раскладки клавиатуры обнаружены в системе. LangSwitcher определяет раскладки автоматически.

Полный список см. в [Поддерживаемые раскладки](/ru/getting-started/layouts).

## Вкладка «Журнал»

- **Включить/выключить логирование** — управление записью преобразований.
- **Лимит записей** — максимальное количество записей или «Безлимитно».
- **Просмотр журнала** — все преобразования с исходным текстом, результатом, раскладками и временем.
- **Оценка** — отмечайте записи как правильные или неправильные.
- **Экспорт JSON** — экспортируйте весь журнал в формате JSON для анализа или обучения ML-моделей.
- **Очистка** — удалить все записи.

::callout{icon="i-lucide-shield-check" color="green"}
Журнал хранится в **локальной базе данных SQLite** только на вашем Mac. Никуда не передаётся.
::
