## Горячая клавиша по умолчанию: двойной Shift

По умолчанию LangSwitcher использует **двойной Shift** (⇧⇧) — нажмите Shift дважды быстро. Это та же горячая клавиша, что и в популярных переключателях раскладок.

Скорость двойного нажатия настраивается в Настройках.

## Пользовательская горячая клавиша

Вы можете записать любую комбинацию клавиш:

1. Откройте **Настройки** → вкладка **Горячая клавиша**
2. Нажмите кнопку **Записать**
3. Нажмите желаемую комбинацию (например, `⌘+Shift+Пробел`)
4. Новая горячая клавиша сохраняется сразу

::callout{icon="i-lucide-alert-triangle" color="amber"}
Избегайте комбинаций, которые конфликтуют с системными или программными горячими клавишами (например, `⌘+C`, `⌘+V`).
::

## Как это работает

Когда срабатывает горячая клавиша:

1. LangSwitcher считывает выделенный текст (или использует умное преобразование для автовыделения)
2. Определяет текущую раскладку и исходную раскладку текста
3. Преобразует каждый символ из исходной раскладки в целевую по карте соответствия
4. Вставляет преобразованный текст через буфер обмена
5. Восстанавливает оригинальное содержимое буфера обмена

Весь процесс занимает миллисекунды и работает в любом приложении.
