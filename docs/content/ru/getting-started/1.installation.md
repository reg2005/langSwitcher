## Системные требования

- **macOS 13** (Ventura) или новее
- Apple Silicon (M1/M2/M3/M4) или Intel
- Разрешение Accessibility (выдаётся при первом запуске)

## Скачать

Перейдите на страницу [GitHub Releases](https://github.com/reg2005/langSwitcher/releases/latest) и скачайте DMG для вашей архитектуры:

| Сборка | Архитектура | Для кого |
|--------|------------|----------|
| **arm64 DMG** | Apple Silicon | Mac с M1, M2, M3, M4 |
| **x86_64 DMG** | Intel | Старые Mac на Intel |
| **Universal DMG** | Обе | Работает на любом Mac |

## Установка

1. Откройте скачанный файл `.dmg`
2. Перетащите **LangSwitcher** в папку **Программы** (Applications)
3. Запустите LangSwitcher из Программ
4. При первом запуске будет запрошено разрешение **Accessibility** — нажмите «Открыть Системные настройки» и включите LangSwitcher

::callout{icon="i-lucide-info" color="blue"}
Если macOS говорит, что приложение «повреждено», см. страницу [Решение проблем](/ru/guide/troubleshooting).
::

## Первые шаги

После установки:

1. LangSwitcher появится как иконка в **строке меню** (вверху справа)
2. Горячая клавиша по умолчанию — **Двойной Shift** (⇧⇧) — нажмите Shift дважды быстро
3. Наберите что-нибудь не в той раскладке, нажмите ⇧⇧ и увидите результат
4. Нажмите на иконку в строке меню для доступа к **Настройкам**, **Журналу преобразований** и другим функциям

::callout{icon="i-lucide-shield" color="amber"}
**Разрешение Accessibility обязательно.** Без него LangSwitcher не сможет читать и заменять текст в других приложениях. Перейдите в **Системные настройки → Конфиденциальность и безопасность → Универсальный доступ** и включите LangSwitcher.
::
