---
title: 'Умное преобразование'
description: 'Как работают режимы умного преобразования — конвертация текста без выделения.'
---

Умное преобразование позволяет конвертировать текст **без ручного выделения**. LangSwitcher анализирует текущую строку и определяет, что нужно преобразовать.

## Режимы

| Режим | Поведение | Для кого |
|-------|----------|----------|
| **Жадная строка** (по умолчанию) | Выделяет до начала строки, сканирует справа налево, находит начало неправильной раскладки, конвертирует только нужную часть | Большинство пользователей |
| **Последнее слово** | Автоматически выделяет последнее слово, конвертирует если оно в неправильной раскладке | Быстрое исправление одного слова |
| **Отключено** | Работает только с явным ручным выделением | Полный ручной контроль |

## Жадная строка (по умолчанию)

Рекомендуемый режим. При нажатии горячей клавиши:

1. LangSwitcher выделяет весь текст от курсора до начала строки
2. Сканирует текст справа налево, ища границу, где начинается неправильная раскладка
3. Преобразует только ошибочную часть
4. Правильно набранный текст не трогает

**Пример:**
```
Вы набрали:   Hello, ghbdtn rfr ltkf
Нажали ⇧⇧:   Hello, привет как дела
```

Английское «Hello, » сохраняется — преобразуется только «абракадабра».

## Последнее слово

Преобразует только последнее набранное слово. Полезно, когда ошибку заметили сразу.

**Пример:**
```
Вы набрали:   Сегодня lheu
Нажали ⇧⇧:   Сегодня друг
```

## Отключено

Умное преобразование выключено. Нужно сначала вручную выделить текст, затем нажать горячую клавишу.

## Изменение режима

Откройте **Настройки** (нажмите иконку в строке меню → Настройки) → вкладка **Основные** → выберите режим умного преобразования из выпадающего списка.
