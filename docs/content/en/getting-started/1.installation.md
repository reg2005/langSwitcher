## System Requirements

- **macOS 13** (Ventura) or later
- Apple Silicon (M1/M2/M3/M4) or Intel
- Accessibility permission (granted on first launch)

## Download

Go to the [GitHub Releases](https://github.com/reg2005/langSwitcher/releases/latest) page and download the DMG for your architecture:

| Build | Architecture | Who is this for? |
|-------|-------------|------------------|
| **arm64 DMG** | Apple Silicon | M1, M2, M3, M4 Macs |
| **x86_64 DMG** | Intel | Older Intel-based Macs |
| **Universal DMG** | Both | Works on any Mac |

## Install

1. Open the downloaded `.dmg` file
2. Drag **LangSwitcher** to the **Applications** folder
3. Launch LangSwitcher from Applications
4. On first launch, you will be prompted to grant **Accessibility** permission — click "Open System Settings" and enable LangSwitcher

::callout{icon="i-lucide-info" color="blue"}
If macOS says the app is "damaged", see the [Troubleshooting](/en/guide/troubleshooting) page.
::

## First Steps

After installation:

1. LangSwitcher appears as an icon in the **menu bar** (top-right of your screen)
2. The default hotkey is **Double Shift** (⇧⇧) — press Shift twice quickly
3. Type something in the wrong layout, press ⇧⇧, and watch it convert
4. Click the menu bar icon to access **Settings**, **Conversion Log**, and more

::callout{icon="i-lucide-shield" color="amber"}
**Accessibility permission is required.** Without it, LangSwitcher cannot read or replace text in other apps. Go to **System Settings → Privacy & Security → Accessibility** and enable LangSwitcher.
::
