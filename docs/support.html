Часто задаваемые вопросы

Приложение не конвертирует текст

Убедитесь, что вы выдали разрешение Accessibility: Системные настройки → Конфиденциальность и безопасность → Универсальный доступ → LangSwitcher.

macOS говорит «приложение повреждено»

Выполните в Терминале: xattr -cr /Applications/LangSwitcher.app

Как изменить горячую клавишу?

Откройте настройки LangSwitcher (клик по иконке в меню-баре → Настройки → вкладка «Горячая клавиша»).

Какие раскладки поддерживаются?

Английская, русская, украинская, немецкая, французская, испанская.

FAQ

The app doesn't convert text

Make sure you granted Accessibility permission: System Settings → Privacy & Security → Accessibility → LangSwitcher.

macOS says the app is "damaged"

Run in Terminal: xattr -cr /Applications/LangSwitcher.app

How to change the hotkey?

Open LangSwitcher settings (click the menu bar icon → Settings → Hotkey tab).

Which keyboard layouts are supported?

English, Russian, Ukrainian, German, French, Spanish.
