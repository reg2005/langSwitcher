FAQ

Зачем LangSwitcher нужно разрешение Accessibility?

LangSwitcher использует macOS Accessibility API для чтения выделенного текста из активного приложения и вставки обратно преобразованного результата. Без этого разрешения приложение не может получить доступ к тексту в других программах. Такой же механизм используют многие текстовые утилиты для macOS.

Для включения: Системные настройки → Конфиденциальность и безопасность → Универсальный доступ → включите LangSwitcher.

macOS говорит, что приложение «повреждено» и не может быть открыто

Это происходит потому, что macOS помещает в карантин приложения, загруженные из интернета не через App Store. Чтобы исправить, откройте Терминал и выполните:

xattr -cr /Applications/LangSwitcher.app

Затем попробуйте открыть приложение снова. Это нужно сделать только один раз. Альтернативно, можно нажать правой кнопкой мыши на приложении и выбрать «Открыть», чтобы обойти Gatekeeper.

Как изменить горячую клавишу?

Откройте настройки LangSwitcher (нажмите на иконку в строке меню → Настройки), перейдите на вкладку Горячая клавиша. Вы можете:

  1. Использовать стандартный Двойной Shift (нажмите Shift дважды быстро)
  2. Записать собственную горячую клавишу, нажав «Записать» и нажав желаемую комбинацию

Какие раскладки клавиатуры поддерживаются?

LangSwitcher в настоящее время поддерживает 6 раскладок:

  1. Английская (U.S. / ABC) — QWERTY
  2. Русская — ЙЦУКЕН
  3. Украинская — ЙЦУКЕН (UA)
  4. Немецкая — QWERTZ
  5. Французская — AZERTY
  6. Испанская — QWERTY (ES)

Раскладки определяются автоматически из системных настроек клавиатуры. В будущих версиях будут добавлены новые раскладки.

Что такое умное преобразование (Smart Conversion)?

Умное преобразование позволяет конвертировать текст без ручного выделения. Доступны три режима:

  1. Жадная строка (Greedy Line) (по умолчанию) — автоматически выделяет до начала строки, находит, где начинается неправильная раскладка, и преобразует только нужную часть
  2. Последнее слово (Last Word) — преобразует только последнее набранное слово
  3. Отключено (Disabled) — работает только с явным выделением текста

Работает ли LangSwitcher без интернета?

Да, полностью. LangSwitcher не делает ни одного сетевого запроса. Всё преобразование выполняется алгоритмически на вашем Mac. Подключение к интернету никогда не требуется.

Как экспортировать журнал преобразований?

Откройте Настройки → вкладка Журнал. Нажмите кнопку «Экспорт JSON». Журнал экспортируется в формате JSON, который можно использовать для анализа или обучения ML-моделей.

Результат преобразования неправильный

Это может происходить, когда текст содержит символы из нескольких раскладок. Попробуйте:

  1. Вручную выделить конкретный текст для преобразования, затем нажать горячую клавишу
  2. Переключить режим умного преобразования в Настройках (попробуйте «Последнее слово» или «Отключено»)
  3. Проверить, что нужные раскладки клавиатуры включены в Системных настройках → Клавиатура → Источники ввода

Если проблема сохраняется, пожалуйста, сообщите о ней на GitHub с исходным текстом и ожидаемым результатом.
