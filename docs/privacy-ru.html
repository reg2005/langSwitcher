Политика конфиденциальности

Дата вступления в силу: 1 марта 2026 г.

🔒

Коротко: LangSwitcher не собирает, не передаёт и не распространяет ваши данные. Вся обработка текста происходит исключительно на вашем Mac. Никакой аналитики, телеметрии и сетевых запросов.

1. Сбор данных

LangSwitcher не собирает никаких персональных данных. А именно:

  • Набранный или преобразованный текст никуда не отправляется
  • Аналитика использования и телеметрия не собираются
  • Отчёты о сбоях не отправляются автоматически
  • Никаких cookies, идентификаторов или любого отслеживания
  • Приложение не делает ни одного сетевого запроса

2. Обработка текста

При нажатии горячей клавиши LangSwitcher считывает выделенный (или автоматически определённый) текст из текущего приложения через macOS Accessibility API. Текст преобразуется между раскладками клавиатуры с помощью чисто алгоритмического, локального вычисления. Преобразованный текст вставляется обратно через системный буфер обмена. Текст ни на каком этапе не покидает ваш Mac.

3. Журнал преобразований

LangSwitcher включает опциональный журнал преобразований для просмотра и оценки качества. Этот журнал:

  • Хранится в локальной базе данных SQLite только на вашем Mac
  • Никуда не передаётся и не синхронизируется
  • Полностью под вашим контролем — вы можете просматривать, экспортировать (JSON) или удалять записи в любое время через Настройки
  • Ведение журнала можно полностью отключить в Настройках

4. Разрешение Accessibility

LangSwitcher требует разрешение macOS Accessibility для работы. Это разрешение используется исключительно для:

  • Чтения выделенного текста из активного приложения (через AXUIElement)
  • Симуляции событий клавиатуры для выделения и замены текста (через CGEvent)
  • Определения текущей раскладки клавиатуры (через TISCopyCurrentKeyboardInputSource)

Разрешение Accessibility никогда не используется для мониторинга набора текста, записи нажатий клавиш или доступа к каким-либо данным помимо необходимых для явно запрошенной вами операции преобразования.

5. Сторонние сервисы

LangSwitcher не использует никаких сторонних сервисов, SDK, аналитических фреймворков или внешних зависимостей. Приложение создано исключительно на системных фреймворках Apple (Swift, SwiftUI, AppKit, Carbon).

6. Хранение данных

LangSwitcher сохраняет только:

  • Пользовательские настройки (горячая клавиша, режим преобразования, язык) — в стандартном macOS UserDefaults
  • Журнал преобразований (если включён) — в локальном файле SQLite внутри sandbox-контейнера приложения

Оба пункта полностью под вашим контролем и удаляются при деинсталляции приложения.

7. Открытый исходный код

LangSwitcher — проект с открытым исходным кодом под лицензией MIT. Вы можете проверить полный исходный код на GitHub, чтобы убедиться в отсутствии сбора данных.

8. Изменения в политике

При обновлении данной политики конфиденциальности изменения будут опубликованы на этой странице с обновлённой датой вступления в силу. Поскольку приложение не делает сетевых запросов, уведомления внутри приложения не будет — проверяйте эту страницу или репозиторий на GitHub.

9. Контакты

Если у вас есть вопросы по данной политике конфиденциальности, откройте issue на GitHub или посетите страницу поддержки.
