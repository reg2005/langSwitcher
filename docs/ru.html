Создан для продуктивности

Всё, чтобы перестать тратить время на перепечатывание текста.

⚡

Мгновенная конвертация

Нажмите горячую клавишу — текст конвертируется за миллисекунды через буфер обмена. Работает в любом приложении.

🧠

Умная конвертация

Не нужно выделять текст. Режим «Жадная строка» сам находит, где начинается неправильная раскладка.

⌨

Двойной Shift

Горячая клавиша по умолчанию: два быстрых нажатия Shift. Можно настроить любую комбинацию.

📊

Журнал конвертаций

Каждая конвертация сохраняется в локальную базу SQLite. Оценивайте как правильно/неправильно. Экспорт в JSON.

🌍

5 раскладок

Английская, русская, немецкая, французская, испанская. Определяются автоматически.

🔒

Приватность

Данные не покидают Mac. Никакой аналитики, телеметрии, сетевых запросов. Открытый код, MIT.

📌

Сохранение пунктуации

Символы ?, !, / остаются неизменными. Конвертируются только символы раскладки.

🛠️

Ноль зависимостей

Чистый Swift, никаких внешних библиотек и словарей. Алгоритмическое определение по позициям клавиш.

✅

93 теста

Умная конвертация

Больше не нужно выделять текст вручную. LangSwitcher сам определяет, что нужно исправить.

РежимПоведениеПример
Жадная строка (по умолч.)Выделяет до начала строки, сканирует справа налево, находит границу неправильной раскладкиghbdtn rfr ltkf → привет как дела
Последнее словоВыделяет только последнее слово перед курсором...lheu → ...друг
ВыключеноРаботает только с ручным выделением текстаВыделите → ⇧⇧ → готово
