Frequently asked questions and how to get help

FAQ

Why does LangSwitcher need Accessibility permission?

LangSwitcher uses the macOS Accessibility API to read selected text from the active application and paste back the converted result. Without this permission, the app cannot access text in other apps. This is the same mechanism used by many macOS text utilities.

To enable: System Settings → Privacy & Security → Accessibility → toggle on LangSwitcher.

macOS says the app is “damaged” and can’t be opened

This happens because macOS quarantines apps downloaded from the internet that are not from the App Store. To fix it, open Terminal and run:

xattr -cr /Applications/LangSwitcher.app

Then try opening the app again. This only needs to be done once. Alternatively, you can right-click the app and choose “Open” to bypass Gatekeeper.

How do I change the hotkey?

Open LangSwitcher settings (click the menu bar icon → Settings), go to the Hotkey tab. You can:

  1. Use the default Double Shift (press Shift twice quickly)
  2. Record a custom hotkey by clicking “Record” and pressing your preferred key combination

What keyboard layouts are supported?

LangSwitcher currently supports 6 keyboard layouts:

  1. English (U.S. / ABC) — QWERTY
  2. Russian — JCUKEN
  3. Ukrainian — JCUKEN (UA)
  4. German — QWERTZ
  5. French — AZERTY
  6. Spanish — QWERTY (ES)

Layouts are auto-detected from your system keyboard settings. More layouts will be added in future versions.

What is Smart Conversion?

Smart Conversion lets you convert text without manually selecting it first. Three modes are available:

  1. Greedy Line (default) — Automatically selects to the start of the line, finds where the wrong layout begins, and converts only the affected portion
  2. Last Word — Converts only the last word you typed
  3. Disabled — Only works with explicit text selection

Does LangSwitcher work offline?

Yes, completely. LangSwitcher makes zero network requests. All conversion is done algorithmically on your Mac. No internet connection is ever required.

How do I export the conversion log?

Open Settings → Log tab. Click the “Export JSON” button. The log is exported as a JSON file that you can use for analysis or ML training data.

The conversion result is incorrect

This can happen when text contains characters from multiple layouts. Try:

  1. Manually selecting the specific text you want to convert, then pressing the hotkey
  2. Switching Smart Conversion mode in Settings (try “Last Word” or “Disabled”)
  3. Checking that the correct keyboard layouts are enabled in System Settings → Keyboard → Input Sources

If the issue persists, please report it on GitHub with the original text and expected result.
