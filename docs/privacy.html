Политика конфиденциальности — LangSwitcher

Дата вступления в силу: 1 марта 2026 г.

Сбор данных

LangSwitcher не собирает, не передает и не хранит никаких персональных данных. Приложение работает полностью локально на вашем Mac.

Доступ к тексту

Приложение получает доступ к выделенному тексту исключительно для конвертации раскладки. Текст обрабатывается локально в оперативной памяти и никуда не отправляется.

Журнал конвертаций

При включении функции «Журнал конвертаций» данные о конвертациях сохраняются в локальную базу данных SQLite на вашем Mac. Эти данные:

  • Хранятся только на вашем устройстве
  • Никуда не передаются
  • Могут быть удалены вами в любой момент
  • Функция логирования отключена по умолчанию

Разрешение Accessibility

Приложение запрашивает разрешение «Универсальный доступ» (Accessibility) в Системных настройках. Это необходимо для:

  • Чтения выделенного текста (Cmd+C)
  • Вставки сконвертированного текста (Cmd+V)
  • Обнаружения горячей клавиши (двойной Shift)

Данное разрешение не используется для каких-либо иных целей.

Сетевые запросы

LangSwitcher не выполняет никаких сетевых запросов. Приложение не содержит аналитики, телеметрии, рекламы или каких-либо средств отслеживания.

Открытый исходный код

Исходный код приложения доступен на GitHub. Вы можете самостоятельно убедиться в отсутствии сбора данных.

Контакты

Если у вас есть вопросы о конфиденциальности, создайте issue на GitHub.

Privacy Policy — LangSwitcher

Effective date: March 1, 2026

Data Collection

LangSwitcher does not collect, transmit, or store any personal data. The application works entirely locally on your Mac.

Text Access

The app accesses selected text solely to convert keyboard layouts. Text is processed locally in memory and is never sent anywhere.

Conversion Log

When the "Conversion Log" feature is enabled, conversion data is saved to a local SQLite database on your Mac. This data:

  • Is stored only on your device
  • Is never transmitted anywhere
  • Can be deleted by you at any time
  • The logging feature is disabled by default

Accessibility Permission

The app requests the Accessibility permission in System Settings. This is required to:

  • Read selected text (Cmd+C)
  • Paste converted text (Cmd+V)
  • Detect the hotkey (double Shift)

This permission is not used for any other purpose.

Network Requests

LangSwitcher makes no network requests whatsoever. The app contains no analytics, telemetry, advertising, or tracking of any kind.

Open Source

The source code is available on GitHub. You can verify the absence of data collection yourself.

Contact

If you have privacy questions, please open an issue on GitHub.
